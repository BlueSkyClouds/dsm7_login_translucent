0、上传脚本到群晖 <br>
1、开启ssh并进入群晖后台 <br>
2、sudo -i  切换Roor用户 <br>
3、cd 《上传脚本的路径》 可在File Station 右键点击install脚本-属性-位置 获取 <br>
4、chmod 755 install.sh 赋予脚本运行权限 <br>
5、./install.sh 运行脚本 <br>
